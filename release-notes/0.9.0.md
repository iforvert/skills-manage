# skills-manage 0.9.0

## English

Cross-platform improvement release focused on Windows support, universal macOS packaging, and reliability fixes across installs, rescans, and source-specific skill detail flows.

### Highlights

- Added **Windows x64** support with `.msi` and portable `.zip` packages
- Upgraded macOS packaging to **universal** `.dmg`, `.zip`, and `.tar.gz` outputs
- Improved Windows/macOS path handling and install reliability
- Preserved Claude source-specific continuity across platform rows, detail views, and explanations
- Shipped rescan refresh fixes plus a round of small UI/accessibility polish

### Packages

- `skills-manage_0.9.0_windows_x64.msi`
- `skills-manage_0.9.0_windows_x64.zip`
- `skills-manage_0.9.0_macos_universal.dmg`
- `skills-manage_0.9.0_macos_universal.zip`
- `skills-manage_0.9.0_macos_universal.tar.gz`

### Notes

- Windows installs can automatically fall back from symlink to copy when symlink creation is unavailable.
- Current desktop builds are unsigned, and macOS notarization is still deferred.
- Checksums should be generated from the final release artifacts before publishing or mirroring builds.

### Changelog

- Added Windows-aware home expansion and cross-platform path rendering in backend commands and frontend views
- Added Windows-friendly install fallback from symlink to copy for skill distribution
- Added GitHub Actions desktop packaging flows for Windows x64 and macOS universal builds
- Fixed Claude multi-source continuity across source-specific rows, detail actions, and explanation loading
- Fixed rescan refresh propagation so central, platform, and discover surfaces stay aligned after full rescans
- Polished settings, sidebar, drawer, and path display interactions with additional accessibility and test coverage

这是一次围绕 Windows 支持、macOS Universal 打包，以及安装/重扫/source-specific 详情链路稳定性修复的跨平台版本。

- 新增 **Windows x64** 支持，可提供 `.msi` 安装包与 `.zip` 便携包
- macOS 打包升级为 **Universal**，可输出 `.dmg`、`.zip`、`.tar.gz`
- 改进 Windows/macOS 场景下的路径处理与安装可靠性
- 优化 Claude 多来源技能连续性，保证平台行、详情视图与解释内容保持 source-specific 一致性
- 修复重扫刷新链路，并补充一批小型 UI / 可访问性细节优化

### 安装包

- `skills-manage_0.9.0_windows_x64.msi`
- `skills-manage_0.9.0_windows_x64.zip`

- `skills-manage_0.9.0_macos_universal.zip`
- `skills-manage_0.9.0_macos_universal.tar.gz`

- 正式发布或镜像分发前，应基于最终产物生成并补充校验和。

### 更新内容

- 为后端命令和前端页面补充 Windows 友好的 home 展开与跨平台路径渲染
